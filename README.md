# Блок Python — Финальный проект по аналитике данных

## Описание
Анализ данных о продажах склада с использованием Python и визуализацией динамики продаж и температуры.

## Данные
- `data.csv` — данные о продажах (Дата, Склад, Контрагент, Номенклатура, Количество)
- `35188.01.01.2018.01.09.2018.1.0.0.ru.ansi.00000000.csv` — данные о погоде в Астане за 2018 год (rp5.ru)

## Задания
1. Загрузка и первичный анализ данных
2. Проверка форматов столбцов
3. Преобразование столбца "Дата" в datetime формат
4. Группировка по дате и подсчёт количества продаж
5. Визуализация динамики продаж
6. Описание графика и выявление аномалий
7. Поиск выброса — дата с минимальными продажами (18 апреля 2018)
8. Топовый товар по средам за июнь-август на складе 3 (product_1 — 2267 шт.)
9. Загрузка данных о погоде, объединение с продажами, визуализация корреляции

## Выводы
- Общий тренд продаж положительный — рост с января по август 2018
- Аномальный спад 18 апреля 2018 — майские праздники
- Наблюдается корреляция между ростом температуры и ростом продаж
- Топ товар на складе 3 по средам летом — product_1

## Инструменты
- Python 3
- pandas, numpy, matplotlib, seaborn
